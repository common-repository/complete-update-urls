Complete Update URLs is a complete URL update solution when moving your WordPress site from one location to another.

== Description ==

Complete Update URLs will update your URLs from the current site location to a new site location. Complete
Update URLs is a *complete* update solution that will update all of your urls from your posts, widgets,
theme configuration, etc. Forget about ever having to manually edit the database again like you ended up
having to do with the other URL update solutions out there. Complete Update URLs is simple and easy.

[Support](http://brightflock.com/complete-update-urls-support) |
[FAQ](http://brightflock.com/complete-update-urls-faqs)

Some benefits:

* Put in your new url and click one button. Can't get any simpler than that!
* Full support for serialized and unserialized meta-data. No database editing required!
* Have an advanced theme that you spent days configuring? Complete Update URLs will update those settings too!
* Find it difficult to move your Wordpress site? Complete Update URL makes it easy.
* Easy command line script also included.

The Complete Update URLs project is sponsored by [Brightflock](http://brightflock.com).

== Installation ==

1. You can either use the automatic plugin installer or your FTP program to upload it to your wp-content/plugins directory
the top-level folder. Don't just upload all the php files and put them in `/wp-content/plugins/`.
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. Visit the Complete Update URLs panel. 
1. Enter your new site location after backing up your WordPress database and click 'Update URLs'.
1. Copy your WordPress database to its new location.
1. That's it! Easy as pie!
